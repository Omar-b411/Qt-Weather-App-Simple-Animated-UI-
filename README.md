🌦 Qt Weather App (Simple Animated UI)

A beginner-friendly weather application built using C++ and Qt Widgets. This project focuses on creating a modern-looking user interface with smooth background animation and basic user interaction.

Users can enter a city name and get a simple weather display (currently using mock data). The project demonstrates how to use Qt for GUI design, painting effects, and timers for animation.

This is a great starting project for learning desktop app development with Qt.
